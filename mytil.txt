- 버전 관리
작업 폴더의 변경 사항이 있을 때 마다 관리
- git 초기화 (initialize, init)
git의 시작
- add / staging
커밋 하고자 하는 대상들을 스테이지에 올려놈.몇 개의 파일만 선택도 가능
- commit (커밋)
버전을 만든다.
- commit 내역(history)
버전들의 내역
- push
로컬repo에서 원격repo로 업로드
- pull
원격repo에서 로컬repo로 다운로드 
- clone
로컬repo가 없을 때 원격repo를 새로운 로컬repo로 그대로 복제 
- tracking
로컬repo와 원격repo를 연결
